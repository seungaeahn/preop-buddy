# PreOp Buddy — 검증 계획서

> 버전: 1.0 | 작성일: 2026-03-13

---

## 1. 검증 개요

PreOp Buddy는 세 가지 핵심 가설을 검증한다. 검증은 **해커톤 당일 즉시 검증** → **단기 파일럿(1개월)** → **장기 임상 검증(3개월)** 3단계로 진행한다.

| 가설 | 검증 단계 | 핵심 지표 |
|---|---|---|
| H1: 수술 이해도 50% 향상 | 단기 파일럿 | 5문항 퀴즈 전/후 점수 |
| H2: 수술 전 불안도 20% 감소 | 단기 파일럿 | VAS 0~10 전/후 점수 |
| H3: AI 정보 정확도 90% 이상 | 해커톤 당일 | 의료진 리뷰 체크리스트 |

---

## 2. 해커톤 당일 즉시 검증 (Day 0)

> 목표: 데모 현장에서 실행 가능한 최소 검증

### 2-1. 전문가 정확도 검토 (H3 검증)

**방법:**
1. 대표 수술 5개 선정 (복강경 담낭절제술, 무릎 반월판 절제술, 충수돌기 절제술, 백내장 수술, 갑상선 절제술)
2. PreOp Buddy로 각 수술 정보 생성 후 출력
3. 의료 지식 보유자(의사/간호사/의대생) 1인이 아래 체크리스트로 검토

**정확도 평가 체크리스트 (수술별 10문항):**

| 항목 | 평가 기준 | 점수 |
|---|---|---|
| 수술 개요 | 핵심 수술 목적/방법이 의학적으로 올바른가 | 0~2 |
| 마취 방법 | 전신/국소 구분이 올바른가 | 0~1 |
| 주요 수술 단계 | 단계 순서가 실제와 일치하는가 | 0~2 |
| 금식 시간 | 표준 지침(8시간)과 일치하는가 | 0~1 |
| 입원 기간 | 일반적 범위 내인가 | 0~1 |
| 회복 기간 | 의학적으로 현실적인가 | 0~1 |
| 주요 합병증 | 누락된 중요 주의사항 없는가 | 0~1 |
| 위험 과장 여부 | 공포 유발 표현 없는가 | 0~1 |

**합격 기준:** 5개 수술 평균 80점 이상 (10점 만점)

**현재 결과 (2026-03-13 자체 검토):**

| 수술명 | 점수 | 특이사항 |
|---|---|---|
| 복강경 담낭절제술 | 9/10 | 포트 위치 설명 생략됨 (비중요) |
| 충수돌기 절제술 | 9/10 | - |
| 백내장 수술 | 8/10 | 국소마취 설명 단순화됨 (적절) |
| 무릎 반월판 절제술 | 9/10 | - |
| 갑상선 절제술 | 8/10 | 신경 손상 위험 언급 미흡 |
| **평균** | **8.6/10** | **86% 정확도 → H3 목표(90%) 근접** |

> ⚠️ 위 수치는 자체 검토 결과이며, 정식 의료진 검토로 보완 예정

---

### 2-2. 현장 사용성 테스트 (데모 참관자 대상)

**방법:** 해커톤 데모 참관자 중 자원자 3~5명에게 아래 절차 진행

**시나리오:**
> "당신은 다음 주에 복강경 담낭절제술을 앞두고 있습니다. PreOp Buddy를 사용해서 수술 준비를 해보세요."

**측정 항목:**

| 항목 | 측정 방법 |
|---|---|
| 태스크 완료율 | 검색 → 결과 확인 → 체크리스트 1개 체크 완료 비율 |
| 시스템 사용성 척도(SUS) | 10문항 설문 (0~100점, 68점 이상 = 허용 가능) |
| 주관적 만족도 | "이 서비스가 수술 준비에 도움이 될 것 같다" (5점 리커트) |
| 불안 완화 체감 | "사용 전보다 수술이 덜 무섭게 느껴진다" (5점 리커트) |

**SUS 설문 10문항:**
1. 이 시스템을 자주 사용하고 싶다
2. 이 시스템이 불필요하게 복잡하다고 느꼈다 (역문항)
3. 이 시스템은 사용하기 쉬웠다
4. 이 시스템을 사용하려면 전문 지식이 필요할 것 같다 (역문항)
5. 이 시스템의 다양한 기능이 잘 통합되어 있다
6. 이 시스템에서 일관성이 없는 부분이 많았다 (역문항)
7. 이 시스템을 빠르게 배울 수 있을 것 같다
8. 이 시스템이 매우 불편하게 느껴졌다 (역문항)
9. 이 시스템을 사용하는 데 자신감을 느꼈다
10. 이 시스템을 사용하기 전에 많은 것을 배워야 했다 (역문항)

**합격 기준:** SUS 68점 이상, 만족도 4점 이상

## 3. 단기 파일럿 검증 (D+7 ~ D+30)

> 목표: H1, H2 가설 정량 검증

### 3-1. 참여자 모집

| 항목 | 내용 |
|---|---|
| 대상 | 수술 예정 환자 또는 최근 수술 경험자 10명 |
| 연령 | 40~70대 |
| 모집 방법 | 지인 네트워크 / 온라인 공고 |
| 보상 | 없음 (자원봉사) |
| 윤리 | 개인정보 비수집, 언제든 철회 가능 |

### 3-2. H1 검증: 수술 이해도 측정

**도구:** 수술별 5문항 이해도 퀴즈

**퀴즈 예시 (복강경 담낭절제술):**

| 번호 | 문항 | 선택지 | 정답 |
|---|---|---|---|
| Q1 | 복강경 수술은 어떤 방식으로 진행되나요? | ①큰 절개 ②작은 구멍 여러 개 ③내시경으로만 | ② |
| Q2 | 수술 전 금식은 몇 시간 전부터 해야 하나요? | ①4시간 ②8시간 ③12시간 | ② |
| Q3 | 퇴원 후 일상 복귀까지 보통 얼마나 걸리나요? | ①1~2일 ②1~2주 ③1~2달 | ② |
| Q4 | 수술 중 어떤 마취를 사용하나요? | ①전신마취 ②국소마취 ③무마취 | ① |
| Q5 | 혈액 희석제를 복용 중이라면? | ①계속 복용 ②수술 전 중단 상담 ③즉시 중단 | ② |

**절차:**
1. PreOp Buddy 사용 **전** 퀴즈 → 사전 점수(A)
2. PreOp Buddy로 해당 수술 정보 열람 (5~10분)
3. 동일 퀴즈 재시행 → 사후 점수(B)
4. 향상율 = (B-A)/A × 100

**H1 검증 기준:** 10명 평균 향상율 ≥ 50%

### 3-3. H2 검증: 수술 전 불안도 측정

**도구:** VAS(Visual Analogue Scale) 0~10점 슬라이더

> "지금 수술에 대해 얼마나 불안하신가요? 0은 전혀 불안하지 않음, 10은 매우 불안함입니다."

**절차:**
1. PreOp Buddy 사용 **전** VAS 측정 → 사전 불안도(X)
2. PreOp Buddy 전체 섹션 열람
3. VAS 재측정 → 사후 불안도(Y)
4. 감소율 = (X-Y)/X × 100

**H2 검증 기준:** 10명 평균 감소율 ≥ 20%

### 3-4. 추가 수집 지표

| 지표 | 측정 방법 | 시점 |
|---|---|---|
| 세션 완료율 | 검색 후 마지막 섹션까지 도달 비율 | 자동 측정 |
| 체크리스트 활용율 | 체크박스 1개 이상 클릭 비율 | 자동 측정 |
| 재방문율 | 동일 사용자 2회 이상 방문 비율 | 쿠키 기반 |
| NPS(순추천지수) | "주변 수술 예정자에게 추천하겠다" (0~10) | 사용 후 설문 |

## 4. 장기 임상 검증 (D+30 ~ D+90)

> 목표: 실제 의료 현장 적용 가능성 검증

### 4-1. 협력 병원 파일럿

| 항목 | 내용 |
|---|---|
| 규모 | 1개 병원, 외래 수술 환자 50명 |
| 기간 | 3개월 |
| 방법 | 수술 전날 PreOp Buddy 태블릿 제공 |
| 대조군 | 기존 동의서만 수령한 환자 50명 |

### 4-2. 임상 측정 지표 추가

| 지표 | 측정 주체 |
|---|---|
| 수술 전 추가 질문 횟수 | 담당 간호사 기록 |
| 수술 취소/연기 비율 (준비 미흡으로 인한) | 병원 기록 |
| 환자 만족도 설문 (수술 후 1일차) | 연구팀 직접 조사 |

## 5. 측정 도구 구현 계획

| 도구 | 구현 방법 | 타임라인 |
