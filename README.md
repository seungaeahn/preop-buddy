# 🏥 PreOp Buddy — 수술 전 환자 교육 AI 어시스턴트

> "수술이 무섭지 않도록, AI가 미리 알려드립니다"

수술명을 입력하면 AI가 **환자 눈높이에 맞춘 수술 설명, 준비사항, 회복 과정, FAQ**를 자동으로 생성해주는 웹 애플리케이션입니다.

## 🔗 배포 URL

> `https://preop-buddy.vercel.app` *(배포 후 업데이트)*

---

## 🤔 왜 만들었나요?

한국에서 수술을 앞둔 환자 대부분은 **수술 동의서에 서명하지만, 내용을 제대로 이해하지 못합니다.** 동의서는 법률·의학 용어로 가득하고, 의료진은 하루 수십 명의 환자에게 같은 설명을 반복할 시간이 없습니다.

환자가 인터넷에서 직접 검색하면? 부정확하거나 공포를 유발하는 정보에 노출됩니다.

**PreOp Buddy는 이 간극을 메웁니다.** AI가 신뢰할 수 있는 수준의 수술 정보를 환자가 이해할 수 있는 언어로 변환해줍니다.

---

## ✨ 핵심 기능

### 1. AI 수술 정보 생성
수술명을 입력하면 5개 섹션의 구조화된 정보를 생성합니다:
- **수술 개요** — 어떤 수술인지 쉬운 비유로 설명
- **수술 과정** — 마취부터 회복실까지 단계별 안내
- **수술 전 준비사항** — 금식, 약물, 준비물 체크리스트
- **회복 과정** — 입원 기간, 일상 복귀 시점, 주의사항
- **자주 묻는 질문** — 환자들이 가장 궁금해하는 5가지

### 2. 난이도 조절
"쉽게" / "자세하게" / "의학 용어 포함" 3단계로 설명 수준을 선택할 수 있습니다.

### 3. 수술 전 체크리스트
준비사항을 인터랙티브 체크리스트로 제공하여 하나씩 확인할 수 있습니다.

## 🛠️ 기술 스택

| 영역 | 기술 | 선택 이유 |
|---|---|---|
| 프론트엔드 | React (Vite) + Tailwind CSS | 빠른 개발 속도, 반응형 UI |
| AI | Claude API (claude-sonnet-4-20250514) | 한국어 품질 우수, JSON 출력 안정적 |
| 배포 | Vercel | Git push → 자동 배포, 무료 |
| 상태관리 | React useState | 별도 백엔드 불필요, 구조 단순화 |

### 아키텍처
```
[사용자] → [React SPA (Vercel)] → [Claude API (streaming)] → [구조화된 수술 정보 UI]
```

백엔드 서버 없이, 프론트엔드에서 Claude API를 직접 호출하는 단순한 구조입니다.

## 🚀 로컬 실행 방법

```bash
# 1. 저장소 클론
git clone https://github.com/{username}/preop-buddy.git
cd preop-buddy

# 2. 의존성 설치
npm install

# 3. 환경 변수 설정
cp .env.example .env
# .env 파일에 VITE_CLAUDE_API_KEY=your_api_key 입력

# 4. 개발 서버 실행
npm run dev
```

`http://localhost:5173`에서 확인할 수 있습니다.

## 📊 검증 계획

### 가설
- **H1:** PreOp Buddy 사용 시, 환자의 수술 이해도가 기존 동의서 대비 **50% 이상 향상**된다
- **H2:** PreOp Buddy 사용 환자의 **수술 전 불안 점수(VAS)가 20% 이상 감소**한다

### 성공 지표
| 지표 | 측정 방법 | 목표 |
|---|---|---|
| 수술 이해도 | 5문항 퀴즈 전/후 비교 | 50% 향상 |
| 수술 전 불안도 | VAS 0~10 설문 | 20% 감소 |
| 정보 정확도 | 의료진 리뷰 (10개 샘플) | 90% 이상 |
| 사용 완료율 | 정보 생성 → 전체 읽기 | 70% 이상 |

자세한 내용은 [PRD](./docs/PRD.md)를 참고하세요.
